Forward models in the cerebellum

Julian Thukral, Julien Vitay and Fred Hamker

The presented model is a reservoir computing model structurally inspired by the human cerebellum. Input (mossy fibres) is fed into a reservoir and the output cells (projection neurons). A general hebbian learning layer is used as an intermediary to decorrelate the input information before it is fed into the reservoir. The reservoir excites the purkinje cells which in turn inhibit the projection neurons. The model is trained to predict the next position of the hand of a 2d arm in circular motion. The base of the arm is situated at the coordinate origin. Training is done with 100.000 Circles. Each differentiating in the center of the circle, the radius, and the starting position of the hand in the circle. Input given to the model consisted of the following: Current position of the hand. Given as x, y coordinates. The movement command. Given as the delta of the current and next position of the angle of the shoulder and elbow joints. The last movement. Given as the delta of the last hand position and the current hand position. (Œîx and Œîy) The input was based on a calculated target circle, not on the last predictions of the model. In short the input fed to the model were for example the x,y-coordinates where the hand should be if the movement was done without any error. Output of the model consists of the x,y coordinates of the predicted position of the hand given the input information. The error is calculated as the normalized mean squared error of the delta between target (calculated next position of the hand) and prediction of the model.
